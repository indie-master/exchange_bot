# Exchange Bot

Telegram-бот для получения и конвертации валютных курсов с использованием API [ExchangeRate-API](https://www.exchangerate-api.com).

## Особенности

- Получение текущих курсов валют (USD, EUR, CNY).
- Конвертация валют.
- Простой и удобный интерфейс.
